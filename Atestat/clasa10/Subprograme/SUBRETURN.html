Subprograme - Funcții care Returnează Valori

În această lecție, analizăm funcții care calculează și returnează valori în diferite formate și utilizări.

Introducere

Funcțiile sunt subprograme care efectuează calcule specifice și returnează rezultatul. Ele pot avea parametri de intrare și întotdeauna un tip de retur definit.

1. Funcția pentru Sumă

suma returnează suma a două sau trei numere.

Cod pentru 2 numere:
int suma(int a, int b){
    return a + b;
}
Cod pentru 3 numere:
int suma(int a, int b, int c){
    return a + b + c;
}

2. Funcția pentru Verificarea Parității

este_par verifică dacă un număr întreg este par.

Cod:
bool este_par(int n){
    return n % 2 == 0;
}

Observație: Returnează true dacă numărul este par, altfel false.

3. Funcția pentru Puteri

a_la_b calculează puterea unui număr întreg a ridicat la o putere b.

Cod:
long long a_la_b(int a, int b){
    long long prod = 1;
    for(int i = 1; i <= b; ++i)
        prod = prod * a;
    return prod;
}

4. Funcția pentru Jumătate

jumatate calculează jumătatea unui număr întreg și returnează rezultatul ca număr zecimal.

Cod:
double jumatate(int n){
    double jum = 0.5 * n;
    return jum;
}

5. Exemplu Practic

Un exemplu complet care utilizează toate funcțiile:

#include <iostream>
using namespace std;

int main(){
    int a = 5, b = 10, c = 3;

    cout << "Suma a + b: " << suma(a, b) << endl;
    cout << "Suma a + b + c: " << suma(a, b, c) << endl;

    cout << "a este par? " << (este_par(a) ? "Da" : "Nu") << endl;

    cout << "a ridicat la b: " << a_la_b(a, b) << endl;

    cout << "Jumatate din a: " << jumatate(a) << endl;

    return 0;
}

6. Observații

  • Funcțiile utilizează tipuri de retur adecvate scopului lor: int, bool, long long, double.
  • Aceste funcții pot fi utilizate pentru a modulariza codul și pentru a evita repetarea logicii.
  • În funcția a_la_b, complexitatea este O(b).
