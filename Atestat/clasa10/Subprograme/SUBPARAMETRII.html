Subprograme

În această lecție, discutăm despre subprograme care întorc valori prin parametri și oferim exemple practice.

Introducere

Subprogramele reprezintă blocuri de cod care realizează operații specifice și pot modifica valorile variabilelor primite prin parametri.

1. Subprogram pentru Adăugare

Subprogramul adaugare primește doi parametri, adaugă valoarea celui de-al doilea parametru la primul și actualizează valoarea acestuia.

Cod:
void adaugare(int &a, int b){
    a += b;
}

2. Subprogram pentru Ștergerea Cifrelor Pare

Subprogramul stergere_cifre_pare elimină cifrele pare dintr-un număr primit prin parametru și returnează rezultatul actualizat.

Cod:
void stergere_cifre_pare(int &n){
    int nr_nou = 0, p = 1;
    while(n != 0){
        int cif = n % 10;
        if(cif % 2 == 1){
            nr_nou = nr_nou + cif * p;
            p *= 10;
        }
        n /= 10;
    }
    n = nr_nou;
}

3. Subprograme pentru Citire

citesteNumere citește două numere întregi, iar citesteNumar citește un singur număr întreg.

Cod:
void citesteNumere(int &a, int &b){
    cin >> a >> b;
}

void citesteNumar(int &n){
    cin >> n;
}

4. Subprogram pentru Afișare

afisareValoare afișează valoarea unui număr întreg primit ca parametru.

Cod:
void afisareValoare(int a){
    cout << a << ' ';
}

5. Exemplu Practic

Un exemplu complet care utilizează toate subprogramele:

Cod:
#include <iostream>
using namespace std;

int main(){
    int a, b, n;

    citesteNumere(a, b); // Citim două numere
    citesteNumar(n);     // Citim un alt număr

    adaugare(a, b);      // Adunăm b la a
    stergere_cifre_pare(n); // Ștergem cifrele pare din n

    afisareValoare(a);   // Afișăm valoarea lui a
    afisareValoare(n);   // Afișăm valoarea lui n

    return 0;
}

6. Observații

  • Toate subprogramele utilizează parametri prin referință pentru a actualiza valorile direct în memoria variabilelor apelante.
  • Subprogramele sunt reutilizabile, ceea ce face codul mai modular și mai clar.
