Subprograme - Prelucrarea și Returnarea Tablourilor

În această lecție, analizăm cum să citim, prelucrăm și returnăm tablouri unidimensionale și bidimensionale folosind funcții.

1. Subprograme pentru Tablouri Unidimensionale

Funcții care efectuează operații de bază asupra vectorilor.

a) Citirea unui Vector

citireVector citește dimensiunea și elementele unui vector.

Cod:
void citireVector(int *a, int &n){
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];
}

b) Suma Elementelor unui Vector

sumaElementeVector calculează suma elementelor dintr-un vector.

Cod:
int sumaElementeVector(int a[], int n){
    int sum = 0;
    for(int i = 1; i <= n; ++i)
        sum += a[i];
    return sum;
}

c) Afișarea unui Vector

afisareVector afișează toate elementele unui vector.

Cod:
void afisareVector(int a[], int n){
    for(int i = 1; i <= n; ++i)
        cout << a[i] << ' ';
}

2. Subprograme pentru Tablouri Bidimensionale

Funcții care prelucrează matrici.

a) Citirea unei Matrice

citireMatrice citește dimensiunile și elementele unei matrice.

Cod:
void citireMatrice(int a[][51], int &n, int &m){
    cin >> n >> m;
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= m; ++j)
            cin >> a[i][j];
}

b) Suma Elementelor unei Matrice

sumaMatrice calculează suma elementelor unei matrice folosind funcția sumaElementeVector.

Cod:
int sumaMatrice(int a[][51], int n, int m){
    int sum = 0;
    for(int i = 1; i <= n; ++i)
        sum += sumaElementeVector(a[i], m);
    return sum;
}

c) Afișarea unei Matrice

afisareMatrice afișează toate elementele unei matrici.

Cod:
void afisareMatrice(int a[][51], int n, int m){
    for(int i = 1; i <= n; ++i, cout << '\\n')
        for(int j = 1; j <= m; ++j)
            cout << a[i][j] << ' ';
}

3. Funcții care Returnează Tablouri

Subprogramele pot returna tablouri utilizând alocare dinamică.

a) Returnarea unui Vector

citireVector returnează un vector alocat dinamic.

Cod:
int* citireVector(int &n){
    cin >> n;
    int *a = new int[n + 1];
    for(int i = 1; i <= n; ++i)
        cin >> a[i];
    return a;
}

b) Returnarea unei Matrici

citireMatrice returnează o matrice alocată dinamic.

Cod:
int** citireMatrice(int &n, int &m){
    cin >> n >> m;
    int **a = new int*[n+1];
    for(int i = 0; i <= n; ++i)
        a[i] = new int[m+1];

    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= m; ++j)
            cin >> a[i][j];

    return a;
}

4. Exemplu Practic

Citirea și afișarea unei matrici:

#include <iostream>
using namespace std;

int main(){
    int n, m, **a;

    a = citireMatrice(n, m);

    for(int i = 1; i <= n; ++i, cout << '\\n')
        for(int j = 1; j <= m; ++j)
            cout << a[i][j] << ' ';

    return 0;
}

5. Observații

  • Funcțiile care returnează tablouri alocate dinamic necesită eliberarea memoriei cu delete[] sau delete[][].
  • Tablourile bidimensionale pot fi utilizate ca pointeri dubli.
  • Reutilizarea funcțiilor crește modularitatea și claritatea codului.
