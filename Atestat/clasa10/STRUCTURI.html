Structuri și Algoritmul lui Lee

Explorăm definiția structurilor, funcțiile asociate și aplicarea algoritmului lui Lee pentru probleme clasice de grafuri și matrice.

⚠️ Atenție: Materia din acestă parte este avansată!

1. Structuri în C++

Structurile sunt tipuri de date personalizate care grupează variabile sub același tip logic.

a) Definirea Structurilor

struct data_calendaristica {
    int zi, luna, an;

    void citire_data(){
        cin >> zi >> luna >> an;
    }

    void afisare_data(){
        cout << zi << "/" << luna << "/" << an << endl;
    }
};

b) Funcții în Structuri

Exemplu: Calculul diferenței în zile între două date:

int diferenta_in_zile(data_calendaristica d1, data_calendaristica d2){
    int zile1 = d1.an * 365 + d1.luna * 30 + d1.zi;
    int zile2 = d2.an * 365 + d2.luna * 30 + d2.zi;
    return abs(zile1 - zile2);
}

2. Algoritmul lui Lee

Algoritmul lui Lee este utilizat pentru a calcula distanțe minime pe o matrice (graf neponderat).

a) Implementarea Algoritmului

void lee(int x, int y){
    Q.push({x, y});
    dist[x][y] = 0;
    while(!Q.empty()){
        pair poz = Q.front();
        Q.pop();
        for(int d = 0; d < 4; ++d){
            int nx = poz.first + dx[d];
            int ny = poz.second + dy[d];
            if(inmat(nx, ny) && a[nx][ny] == 0 && dist[nx][ny] == -1){
                dist[nx][ny] = dist[poz.first][poz.second] + 1;
                Q.push({nx, ny});
            }
        }
    }
}

b) Aplicație: Determinarea Numărului de Insule

void lee_insule(int i, int j, int id){
    Q.push({i, j});
    b[i][j] = id;
    while(!Q.empty()){
        pair poz = Q.front();
        Q.pop();
        for(int d = 0; d < 4; ++d){
            int nx = poz.first + dx[d];
            int ny = poz.second + dy[d];
            if(inmat(nx, ny) && a[nx][ny] == 1 && b[nx][ny] == 0){
                b[nx][ny] = id;
                Q.push({nx, ny});
            }
        }
    }
}

c) Exerciții Practice

  • Implementați algoritmul pentru a găsi cea mai mare insulă.
  • Extindeți algoritmul pentru matrice tridimensionale.

3. Observații

  • Structurile: Simplifică gestionarea datelor complexe.
  • Lee: Asigură găsirea celei mai scurte distanțe într-un graf neponderat.
  • Complexitate: O(n * m) pentru o matrice de dimensiuni n x m.
