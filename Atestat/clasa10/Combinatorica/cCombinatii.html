Combinatorică

Concepte fundamentale precum aranjamente, permutări și combinări, explicate cu formule și implementări în C++.

1. Aranjamente

Aranjamentele reprezintă submulțimi ordonate de k elemente selectate dintr-o mulțime de n elemente.

Formula:

Akn = n! / (n-k)! = (n-k+1) * ... * n

Implementare C++:

int Aranjamente(int n, int k){
    int prod = 1;
    for(int i = n - k + 1; i <= n; ++i)
        prod *= i;
    return prod;
}

2. Permutări

Permutările sunt aranjamentele tuturor elementelor dintr-o mulțime. O permutare este o rearanjare a elementelor.

Formula:

Pn = n!

Implementare Recursivă:

int Permutari(int n){
    if(n <= 1)
        return 1;
    return n * Permutari(n - 1);
}

3. Combinări

Combinările reprezintă submulțimi neordonate de k elemente selectate dintr-o mulțime de n elemente.

Formula:

Ckn = Akn / k! = n! / (k!(n-k)!)

Formula Recurentă (Triunghiul lui Pascal):

Ckn = Ck-1n-1 + Ckn-1

Implementare C++:

int Combinari1(int n, int k){
    if(k == 0 || k == n)
        return 1;
    return Combinari1(n-1, k-1) + Combinari1(n-1, k);
}

4. Triunghiul lui Pascal

Triunghiul lui Pascal este o reprezentare grafică a coeficienților binomiali:

        1
       1 1
      1 2 1
     1 3 3 1
    1 4 6 4 1

Generare în C++:

void generareTriunghiPascal(int n){
    for(int i = 0; i <= n; ++i)
        for(int j = 0; j <= i; ++j)
            if(i == j || j == 0)
                C[i][j] = 1;
            else C[i][j] = C[i-1][j-1] + C[i-1][j];
}

5. Submulțimi

Numărul total de submulțimi al unei mulțimi cu n elemente este 2^n.

Implementare C++:

int NrSubmultimi(int n){
    return 1 << n; // 2^n
}

6. Observații

  • Funcțiile iterative sunt mai rapide decât cele recursive, dar ocupă mai mult cod.
  • Rezultatele intermediare pot cauza overflow, motiv pentru care este recomandată utilizarea funcțiilor optimizate.
  • Principiul memorizării poate accelera calculul combinărilor.
