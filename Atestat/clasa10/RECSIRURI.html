Generarea de Șiruri și Matrici Recurente

Descoperă cum să generezi modele recurente folosind recursivitate, inclusiv șiruri "fierăstrău" și matrici cu modele definite.

1. Generarea unui Model în Matrice

generare_pattern umple o matrice cu un model pe baza recursivității. Matricea este împărțită în patru sub-matrici, iar doar prima sub-matrice este umplută cu valoarea 1.

Cod:
void generare_pattern(int i1, int j1, int i2, int j2){
    if(i1 == i2)
        return;
    int miji = (i1 + i2) / 2;
    int mijj = (j1 + j2) / 2;
    for(int i = i1; i <= miji; ++i)
        for(int j = j1; j <= mijj; ++j)
            a[i][j] = 1;
    generare_pattern(i1, mijj+1, miji, j2);
    generare_pattern(miji+1, j1, i2, mijj);
    generare_pattern(miji+1, mijj+1, i2, j2);
}

2. Generarea unui Model cu Valori Descrescătoare

generare_pattern1 umple o matrice cu valori descrescătoare, pornind din centru spre margini, folosind recursivitate.

Cod:
void generare_pattern1(int i1, int j1, int i2, int j2, int val){
    if(val == 0)
        return;
    int miji = (i1 + i2) / 2;
    int mijj = (j1 + j2) / 2;
    a[miji][mijj] = val;
    generare_pattern1(i1, j1, miji-1, mijj-1, val-1);
    generare_pattern1(i1, mijj+1, miji-1, j2, val-1);
    generare_pattern1(miji+1, j1, i2, mijj-1, val-1);
    generare_pattern1(miji+1, mijj+1, i2, j2, val-1);
}

3. Generarea unui Șir de Tip "Fierăstrău"

afisare_fierastrau generează și afișează un șir recursiv de tip "fierăstrău".

Cod:
void afisare_fierastrau(int n){
    if(n == 0)
        return;
    afisare_fierastrau(n - 1);
    cout << n << ' ';
    afisare_fierastrau(n - 1);
}

Exemplu: Pentru n = 3, ieșirea este: 1 2 1 3 1 2 1.

4. Exemplu Practic

Generarea unui model cu valori descrescătoare în matrice:

#include <iostream>
using namespace std;

int a[1025][1025], n;

void generare_pattern1(int i1, int j1, int i2, int j2, int val){
    if(val == 0)
        return;
    int miji = (i1 + i2) / 2;
    int mijj = (j1 + j2) / 2;
    a[miji][mijj] = val;
    generare_pattern1(i1, j1, miji-1, mijj-1, val-1);
    generare_pattern1(i1, mijj+1, miji-1, j2, val-1);
    generare_pattern1(miji+1, j1, i2, mijj-1, val-1);
    generare_pattern1(miji+1, mijj+1, i2, j2, val-1);
}

int main(){
    cin >> n;
    int m = (1 << n) - 1; // Calculăm dimensiunea matricei

    generare_pattern1(1, 1, m, m, n);

    for(int i = 1; i <= m; ++i, cout << '\\n')
        for(int j = 1; j <= m; ++j)
            cout << a[i][j] << ' ';
    return 0;
}

5. Observații

  • Matricea generată are dimensiunea (2^n - 1) x (2^n - 1).
  • Recursivitatea permite generarea de modele complexe într-un mod elegant și intuitiv.
  • Complexitatea depinde de nivelul de împărțire a matricei și poate fi exprimată în funcție de O(n).
