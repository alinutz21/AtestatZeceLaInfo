Recursivitate Generală

Descoperă cum funcționează recursivitatea, aplicațiile sale și exemple de funcții recursive.

1. Ce este Recursivitatea?

Recursivitatea este un proces prin care o funcție se apelează pe ea însăși pentru a rezolva o problemă mai mică.

2. Exemple de Funcții Recursive

a) Număr de Cifre
int nrCif(int n){
    if(n <= 9)
        return 1;
    return 1 + nrCif(n / 10);
}

Model matematic: nrCif(n) = 1 dacă n <= 9, altfel 1 + nrCif(n / 10).

b) Fibonacci
int fibo(int n){
    if(n <= 1)
        return 1;
    return fibo(n - 1) + fibo(n - 2);
}

Generează seria Fibonacci.

c) Factorial
int fact(int n){
    if(n <= 1)
        return 1;
    return n * fact(n - 1);
}

Calculează factorialul unui număr: fact(n) = n * fact(n-1).

d) Puteri
int alab(int a, int b){
    if(b == 0)
        return 1;
    return a * alab(a, b-1);
}

Calculează a^b utilizând metoda recursivă.

3. Recursivitate pentru Prelucrarea Vectorilor

a) Citirea unui Vector
void citireVector(int a[], int n){
    if(n == 0)
        return;
    citireVector(a, n-1);
    cin >> a[n];
}
b) Afișarea unui Vector
void afisareVector(int a[], int n){
    if(n == 0)
        return;
    afisareVector(a, n-1);
    cout << a[n] << ' ';
}

4. Exponențiere Rapidă

a) Metodă Iterativă
int expRapidaIterativa(int a, int b){
    int prod = 1;
    while(b != 0){
        if(b % 2 == 1)
            prod *= a;
        a *= a;
        b /= 2;
    }
    return prod;
}

Complexitate: O(log2(b)).

b) Metodă Recursivă
int expRapidRecursiva(int a, int b){
    if(b == 0)
        return 1;
    int m = expRapidRecursiva(a, b/2);
    if(b % 2 == 0)
        return m * m;
    return m * m * a;
}

Complexitate: O(log2(b)).
