Cifrele unui număr

În această secțiune, vom explora diverse funcții care manipulează cifrele unui număr, cum ar fi numărul de cifre, suma cifrelor, oglindirea, palindromurile și altele. Vom utiliza atât funcții iterativ-ciclice (bucla while) cât și funcții recursive.

1. Numărul de cifre al unui număr

Pentru a obține numărul de cifre ale unui număr, putem folosi o funcție iterativă sau una recursivă:

Funcție Iterativă:
int nr_cif(int n){
    int cnt = 0;
    do{
        cnt++;
        n /= 10;
    }while(n != 0);
    return cnt;
}
Funcție Recursivă:
int nr_cif_rec(int n){
    if(n <= 9)
        return 1;
    return 1 + nr_cif_rec(n / 10);
}

!! ATENȚIE !!

La funcția nr_cif, trebuie să fim atenți la cazul în care n este 0 la apelul inițial. Dacă introducem 0, bucla do-while va executa cel puțin o dată, dar în final, valoarea returnată va fi 0, ceea ce nu este corect pentru numărul 0, deoarece numărul 0 are o singură cifră. Ar trebui să gestionăm acest caz separat.

2. Suma cifrelor unui număr

Putem obține suma cifrelor unui număr folosind o funcție iterativă sau una recursivă:

Funcție Iterativă:
int sum_cif(int n){
    int sum = 0;
    while(n != 0){
        int cif = n % 10;
        sum += cif;
        n /= 10;
    }
    return sum;
}
Funcție Recursivă:
int sum_cif_rec(int n){
    if(n == 0)
        return 0;
    return n % 10 + sum_cif_rec(n / 10);
}

3. Oglindirea unui număr

O funcție care inversează cifrele unui număr, de exemplu pentru 1234 va returna 4321:

int oglindit(int n){
    int ogl = 0;
    while(n != 0){
        int cif = n % 10;
        ogl = ogl * 10 + cif;
        n /= 10;
    }
    return ogl;
}

De asemenea, este important de reținut că:
oglindit(oglindit(n)) nu este întotdeauna egal cu n. Exemplu:
oglindit(oglindit(1230)) = oglindit(321) = 123 != 1230

4. Verificarea dacă un număr este palindrom

Un număr este palindrom dacă se citește la fel de la stânga la dreapta și de la dreapta la stânga. Funcția de mai jos verifică dacă un număr este palindrom:

bool palindrom(int n){
    return n == oglindit(n);
}

5. Ștergerea cifrelor pare dintr-un număr

Această funcție elimină toate cifrele pare dintr-un număr:

int stergere_cifre_pare(int n){
    int nr_nou = 0, p = 1;
    while(n != 0){
        int cif = n % 10;
        if(cif % 2 != 0){
            nr_nou = nr_nou + cif * p;
            p *= 10;
        }
        n /= 10;
    }
    return nr_nou;
}

6. Dublarea cifrelor pare

Funcția de mai jos dublează fiecare cifră pară într-un număr:

int dublare_cifre_pare(int n){
    int nr_nou = 0, p = 1;
    while(n != 0){
        int cif = n % 10;
        if(cif % 2 == 0){
            nr_nou = nr_nou + cif * p;
            p *= 10;
        }
        nr_nou = nr_nou + cif * p;
        p *= 10;
        n /= 10;
    }
    return nr_nou;
}

7. Interschimbarea cifrelor

Această funcție interschimbă perechi de cifre consecutive într-un număr:

int interschimbare_cifre(int n){
    int nr_nou = 0, p = 1;
    while(n != 0){
        int cif1 = n % 10;
        int cif2 = (n / 10) % 10;
        nr_nou = nr_nou + cif2 * p;
        p *= 10;
        nr_nou = nr_nou + cif1 * p;
        p *= 10;
        n /= 100;
    }
    return nr_nou;
}

8. Interschimbarea cifrelor (varianta recursivă)

O variantă recursivă a funcției de interschimbare a cifrelor:

int interschimbare_cifre_rec(int n){
    if(n == 0)
        return 0;
    int cif1 = n % 10;
    int cif2 = (n / 10) % 10;
    return interschimbare_cifre_rec(n / 100) * 100 + cif1 * 10 + cif2;
}
