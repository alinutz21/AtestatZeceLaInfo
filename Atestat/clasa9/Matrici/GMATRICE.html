Generarea Matricelor

În această secțiune, explorăm algoritmi pentru generarea de matrici cu modele specifice, inclusiv distanțe față de diagonale și spirale.

1. Matrice cu distanțe față de diagonala secundară

Fiecare element al matricei este egal cu distanța sa față de diagonala secundară.

Exemplu: Pentru n = 5, matricea este:

4 3 2 1 0
3 2 1 0 1
2 1 0 1 2
1 0 1 2 3
0 1 2 3 4
Cod:
void generare1(int a[][51], int n){
    for(int i = 0; i < n; ++i)
        for(int j = 0; j < n; ++j)
            a[i][j] = abs(n - 1 - i - j);
}

2. Matrice generată în spirală

Elementele matricei sunt dispuse în spirală, în sensul acelor de ceasornic, de la 1 până la n^2.

Exemplu: Pentru n = 4, matricea este:

1  2  3  4
12 13 14  5
11 16 15  6
10  9  8  7
Cod:
void generare2(int a[][51], int n){
    int cnt = 1, i = 1, j = 1;
    while(cnt <= n * n){
        while(a[i][j] == 0 && j <= n && cnt <= n * n)
            a[i][j] = cnt++, j++;
        j--;
        i++;
        while(a[i][j] == 0 && i <= n && cnt <= n * n)
            a[i][j] = cnt++, i++;
        i--;
        j--;
        while(a[i][j] == 0 && j >= 1 && cnt <= n * n)
            a[i][j] = cnt++, j--;
        i--;
        j++;
        while(a[i][j] == 0 && i >= 1 && cnt <= n * n)
            a[i][j] = cnt++, i--;
        i++;
        j++;
    }
}

Exemplu Practic

Un exemplu de generare și afișare a unei matrice spiralate:

#include <iostream>
using namespace std;

int main(){
    int a[51][51], n;
    cin >> n;

    generare2(a, n);

    for(int i = 1; i <= n; ++i, cout << '\\n')
        for(int j = 1; j <= n; ++j)
            cout << a[i][j] << ' ';

    return 0;
}

Observații

  • Complexitatea algoritmului pentru ambele generări este O(n2).
  • Funcțiile sunt adaptate pentru matrice de dimensiuni mici și medii, până la maxim 50x50 în acest exemplu.
  • Generarea în spirală necesită verificări suplimentare pentru evitarea suprascrierilor.
