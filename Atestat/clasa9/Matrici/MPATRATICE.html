Matrici Pătratice

În această secțiune, explorăm concepte legate de diagonale, zone și simetrie în matricile pătratice.

Definiție

O matrice pătratică este o matrice în care numărul de linii este egal cu numărul de coloane (n == m).

int a[101][101]; // Matrice pătratică 101x101

1. Diagonalele unei Matrici

O matrice pătratică are două diagonale principale:

  • Diagonala principală: Elemente pentru care i == j.
  • Diagonala secundară: Elemente pentru care i + j == n + 1 (indexare de la 1) sau i + j == n - 1 (indexare de la 0).

Exemplu:

n = 4
Matricea:
x 0 0 x
0 x x 0
0 x x 0
x 0 0 x

2. Zonele Matricii

Matricea poate fi împărțită în 4 zone față de diagonalele principale:

  • ZONA 1: Deasupra diagonalei principale și deasupra diagonalei secundare.
  • ZONA 2: Deasupra diagonalei principale și sub diagonala secundară.
  • ZONA 3: Sub diagonala principală și sub diagonala secundară.
  • ZONA 4: Sub diagonala principală și deasupra diagonalei secundare.
int s = 0;
// Calculul sumei elementelor din Zona 1 (indexare de la 1)
for(int i = 1; i <= n / 2; ++i)
    for(int j = i + 1; j < n - i + 1; ++j)
        s += a[i][j];

3. Simetria în Matrice

Simetria poate fi determinată față de diverse elemente:

  • Linia mediană: a[i][j] -> a[n-i+1][j] (indexare de la 1).
  • Coloana mediană: a[i][j] -> a[i][n-j+1] (indexare de la 1).
  • Diagonala principală: a[i][j] -> a[j][i].
  • Diagonala secundară: a[i][j] -> a[n-j+1][n-i+1] (indexare de la 1).

4. Determinarea Elementelor de Pe Diagonale

Numărul total de elemente care aparțin ambelor diagonale:

int nr_diagonale = 2 * n - (n % 2);

5. Exemplu Practic

Un exemplu complet care include citirea unei matrici și calcularea sumei elementelor dintr-o zonă:

#include <iostream>
using namespace std;

int main(){
    int a[101][101], n;
    cin >> n;
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= n; ++j)
            cin >> a[i][j];

    int s = 0;
    for(int i = 1; i <= n / 2; ++i)
        for(int j = i + 1; j < n - i + 1; ++j)
            s += a[i][j];

    cout << "Suma Zonei 1: " << s << endl;
    return 0;
}
