Matrici și Tablouri Bidimensionale

Explorează bazele lucrului cu matrici în C++, incluzând exemple și observații despre memorie și funcționare.

Introducere

O matrice în programare este un tablou bidimensional care poate fi privit ca un set de vectori. Fiecare linie a unei matrici este, de fapt, un vector, iar principiile aplicate vectorilor sunt valabile și aici.

1. Declarație și Inițializare

O matrice de dimensiuni maxime 101x101 poate fi declarată astfel:

int a[101][101]; // 101 linii și 101 coloane

Exemplu: Inițializarea unei matrici:

cin >> n >> m; // n = linii, m = coloane
for(int i = 1; i <= n; ++i)
    for(int j = 1; j <= m; ++j)
        cin >> a[i][j];

2. Afișarea unei Matrici

Funcția de afișare folosește liniile matricei ca vectori:

void afisare_vector(int a[], int n){
    for(int i = 1; i <= n; ++i)
        cout << a[i] << ' ';
}

Apelarea acestei funcții pentru fiecare linie:

for(int i = 1; i <= n; ++i, cout << '\\n')
    afisare_vector(a[i], m);

3. Observații despre Memorie

  • Matricile sunt stocate ca vectori de vectori în memorie.
  • Distanța în memorie între două linii consecutive depinde de numărul de coloane.
  • Toate regulile aplicate vectorilor se aplică și matricilor.

4. Exemplu Practic

Un exemplu simplu de citire și afișare a unei matrice:

#include <iostream>
using namespace std;

int main(){
    int a[101][101], n, m;
    cin >> n >> m;
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= m; ++j)
            cin >> a[i][j];

    for(int i = 1; i <= n; ++i, cout << '\\n')
        for(int j = 1; j <= m; ++j)
            cout << a[i][j] << ' ';
    return 0;
}

5. Idei de Extindere

  • Adăugarea unor funcții pentru transpunerea matricelor.
  • Calcularea sumei elementelor de pe o diagonală.
  • Determinarea elementului maxim din matrice.
  • Compararea a două matrici.
