Elemente de baza a limbajului C++

Ce este C++?

C++ este un limbaj de programare de nivel înalt, folosit pentru a dezvolta aplicații software de orice tip, de la aplicații desktop la jocuri și sisteme complexe. C++ este un limbaj cu tipuri de date statice și permite manipularea directă a memoriei, fiind utilizat pentru aplicații de performanță înaltă.

Elementele de bază ale limbajului C++

C++ este un limbaj care include o varietate de elemente de bază care sunt esențiale pentru orice program. Acestea includ variabile, structuri de control, și bucle, printre altele. Iată câteva dintre elementele de bază în C++:

Structuri liniare

În C++, variabilele pot fi declarate folosind tipuri de date fundamentale, precum int, bool, char, etc. Exemplu:

#include <iostream>
#include <fstream>
#include <cmath>
#include <cstring>
using namespace std;

int main(){
    int a, b, c; // [-2^31, 2^31-1] --> număr intreg
    bool gasit = false; // true / false (1 / 0)
    char ch; // stochează un caracter
    float numar_rational;
    double numar_rational2;
    
    a += c;
    a--;
    b -= a;
    gasit != gasit;
    a++;
    a = a % b; // a primește restul împărțirii
}

Structuri alternative

C++ include instrucțiuni de control al fluxului, precum if, else, și switch, care permit programului să execute diferite blocuri de cod în funcție de condiții specifice.

if(a % 2 == 0)
    cout << "a este par";
else if(a % 10 == 3)
    cout << "a are ultima cifra 3";
else
    cout << "a este impar si nu are ultima cifra 3";

Structuri repetitive

Structurile repetitive, cum ar fi for, while, și do-while, sunt folosite pentru a repeta un bloc de cod de mai multe ori, în funcție de o condiție dată. Exemplu:

for(int i = 1; i <= a; i++){
    if(i % 2 == 0)
        continue;
    cout << i << ' ';
    if(i == 5)
        break;
}

int j = 1;
while(j <= a){
    if(j % 2 == 0){
        j++;
        continue ;
    }
    cout << j << ' ';
    if(j == 5)
        break;
    j++;
}

int k = 1;
do{
    if(k % 2 == 0){
        k++;
        continue;
    }
    cout << k << ' ';
    if(k == 5)
        break;
}while(k <= a);

!! ATENȚIE !!

La diferență de buclele for și while, bucla do-while se va executa cel puțin o dată, chiar dacă condiția de oprire este falsă la prima verificare. Acest comportament se datorează faptului că în cazul unui do-while, condiția este verificată abia după ce blocul de cod a fost executat cel puțin o dată.

De exemplu, chiar dacă variabila k are valoarea 6 și condiția k <= a este falsă, bucla se va executa o dată înainte de a verifica condiția. Acesta este un aspect important de reținut atunci când utilizăm do-while la subiectul II de la bac.
