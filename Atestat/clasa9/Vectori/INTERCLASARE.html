Interclasare și Operații pe Mulțimi

Introducere

Algoritmii descriși aici tratează interclasarea a două șiruri ordonate și implementarea unor operații pe mulțimi matematice, cum ar fi reuniunea, intersecția, diferența și diferența simetrică.

1. Interclasare

Descriere: Combinația a două șiruri ordonate într-unul singur, păstrând ordinea elementelor.

Cod:
void interclasare(int a[], int n, int b[], int m, int c[], int &indc){
    indc = 0;
    int inda = 1, indb = 1;
    while(inda <= n && indb <= m){
        if(a[inda] <= b[indb])
            c[++indc] = a[inda++];
        else c[++indc] = b[indb++];
    }
    while(inda <= n)
        c[++indc] = a[inda++];
    while(indb <= m)
        c[++indc] = b[indb++];
}

2. Reuniunea

Descriere: Unirea a două mulțimi pentru a obține toate elementele distincte din ambele mulțimi.

Cod:
void reuniune(int a[], int n, int b[], int m, int c[], int &indc){
    indc = 0;
    int inda = 1, indb = 1;
    while(inda <= n && indb <= m){
        if(a[inda] < b[indb])
            c[++indc] = a[inda++];
        else if(a[inda] > b[indb])
            c[++indc] = b[indb++];
        else
            c[++indc] = a[inda++], indb++;
    }
    while(inda <= n)
        c[++indc] = a[inda++];
    while(indb <= m)
        c[++indc] = b[indb++];
}

3. Intersecția

Descriere: Determină elementele comune din două mulțimi.

Cod:
void intersectie(int a[], int n, int b[], int m, int c[], int &indc){
    indc = 0;
    int inda = 1, indb = 1;
    while(inda <= n && indb <= m){
        if(a[inda] < b[indb])
            inda++;
        else if(a[inda] > b[indb])
            indb++;
        else c[++indc] = a[inda++], indb++;
    }
}

4. Diferența

Descriere: Obține elementele care apar într-o mulțime, dar nu și în cealaltă.

Cod:
void diferenta(int a[], int n, int b[], int m, int c[], int &indc){
    indc = 0;
    int inda = 1, indb = 1;
    while(inda <= n && indb <= m){
        if(a[inda] < b[indb])
            c[++indc] = a[inda++];
        else if(a[inda] > b[indb])
            indb++;
        else inda++, indb++;
    }
    while(inda <= n)
        c[++indc] = a[inda++];
}

5. Diferența Simetrică

Descriere: Reunește elementele care apar în exact una dintre cele două mulțimi.

Cod:
void diferenta_simetrica(int a[], int n, int b[], int m, int c[], int &indc){
    indc = 0;
    int inda = 1, indb = 1;
    while(inda <= n && indb <= m){
        if(a[inda] < b[indb])
            c[++indc] = a[inda++];
        else if(a[inda] > b[indb])
            c[++indc] = b[indb++];
        else inda++, indb++;
    }
    while(inda <= n)
        c[++indc] = a[inda++];
    while(indb <= m)
        c[++indc] = b[indb++];
}

6. Observații

  • Aceste algoritmi presupun că mulțimile sunt reprezentate sub formă de șiruri ordonate.
  • Complexitatea tuturor operațiilor este O(n + m), unde n și m sunt dimensiunile celor două mulțimi.
