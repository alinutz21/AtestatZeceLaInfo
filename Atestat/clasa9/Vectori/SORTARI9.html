Sortarea Vectorilor - O(n2)

Metode de sortare în C++

Acest articol detaliază implementările pentru Selection Sort, Bubble Sort și Insertion Sort. Fiecare metodă este prezentată în variante iterative și recursive.

Selection Sort

Descriere: Algoritmul caută minimul din partea nesortată a vectorului și îl plasează la început. Repetă procesul pentru fiecare poziție.

Cod Iterativ Direct:
void selectionSortIterativDirect(int a[], int n){
    for(int i = 1; i < n; ++i){
        int imin = i;
        for(int j = i + 1; j <= n; ++j)
            if(a[imin] > a[j])
                imin = j;
        swap(a[i], a[imin]);
    }
}
Cod Recursiv:
void selectionSortRecursiv1(int a[], int n){
    if(n <= 1)
        return ;
    for(int i = 1; i < n; ++i)
        if(a[i] > a[n])
            swap(a[i], a[n]);
    selectionSortRecursiv1(a, n-1);
}

Bubble Sort

Descriere: Algoritmul compară fiecare pereche de elemente adiacente și le interschimbă dacă sunt în ordine greșită. Procesul continuă până când vectorul devine sortat.

Cod Iterativ Clasic:
void bubbleSortClasicIterativ(int a[], int n){
    bool este_sortat = false;
    while(!este_sortat){
        este_sortat = true;
        for(int i = 1; i < n; ++i)
            if(a[i] > a[i+1]){
                swap(a[i], a[i+1]);
                este_sortat = false;
            }
    }
}
Cod Recursiv:
void bubbleSortClasicRecursiv(int a[], int n){
    bool este_sortat = true;
    for(int j = 1; j < n; ++j)
        if(a[j] > a[j+1]){
            swap(a[j], a[j+1]);
            este_sortat = false;
        }
    if(!este_sortat)
        bubbleSortClasicRecursiv(a, n);
}

Insertion Sort

Descriere: Algoritmul inserează fiecare element în poziția corectă în porțiunea deja sortată a vectorului.

Cod Iterativ:
void insertionSortIterativ(int a[], int n){
    for(int i = 2; i <= n; ++i){
        int poz = i;
        for(int j = i - 1; j >= 1; --j)
            if(a[j] > a[i])
                poz = j;
        int aux = a[i];
        for(int j = i - 1; j >= poz; --j)
            a[j+1] = a[j];
        a[poz] = aux;
    }
}
Cod Recursiv:
void insertionSortRecursiv(int a[], int n, int i = 2){
    if(i > n)
        return ;
    for(int j = i; j > 1; --j)
        if(a[j] < a[j-1])
            swap(a[j], a[j-1]);
    insertionSortRecursiv(a, n, i+1);
}

Observații Generale

  • Complexitatea algoritmilor este O(n2), ceea ce îi face potriviți doar pentru vectori mici.
  • Implementările recursive sunt mai elegante, dar pot consuma mai multă memorie datorită stivei de apeluri.
  • Fiecare algoritm prezintă avantaje și dezavantaje, iar alegerea depinde de situația specifică.
