Vectori Caracteristici și de Frecvență

Introducere

În această secțiune, vom explora utilizarea vectorilor pentru stocarea caracteristicilor și frecvențelor, alături de algoritmi asociați cum ar fi Counting Sort și Ciurul lui Eratostene.

1. Vectori Caracteristici

Vectorii caracteristici stochează informații despre proprietăți specifice ale valorilor.

bool C[1000001]; // Exemplu: C[x] = true dacă x respectă proprietatea

Exemplu: Calcularea numărului de cifre distincte într-un șir.

bool C[10];
int nr_cif_distincte(int a[], int n){
    for(int i = 1; i <= n; ++i)
        C[a[i]] = true;
    int cnt = 0;
    for(int i = 0; i <= 9; ++i)
        if(C[i])
            cnt++;
    return cnt;
}

2. Vectori de Frecvență

Vectorii de frecvență stochează numărul de apariții ale fiecărei valori într-un șir.

int F[1000001]; // Exemplu: F[x] = numărul de apariții ale lui x

Exemplu: Counting Sort

void countingSort(int a[], int n){
    for(int i = 1; i <= n; ++i)
        F[a[i]]++;
    n = 0;
    for(int i = 0; i <= 1000000; ++i)
        while(F[i] != 0)
            a[++n] = i, F[i]--;
}

3. Ciurul lui Eratostene

Algoritmul clasic pentru calcularea numerelor prime până la un număr n.

bool ciur[1000001];
int prime[80001], nr_prime = 0;

void erathostene(int n){
    ciur[1] = ciur[0] = true;
    for(int i = 2; i * i <= n; ++i)
        if(!ciur[i])
            for(int j = i; i * j <= n; ++j)
                ciur[i * j] = true;
    for(int i = 2; i <= n; ++i)
        if(!ciur[i])
            prime[++nr_prime] = i;
}

4. Factorizarea Numerelor

Utilizând Ciurul lui Eratostene, putem efectua factorizarea eficientă a unui număr.

void factorizare_ciur(int n){
    int indp = 1;
    while(n > 1){
        int p = 0;
        while(n % prime[indp] == 0)
            n /= prime[indp], p++;
        if(p)
            cout << prime[indp] << '^' << p << '\\n';
        indp++;
        if(prime[indp] * prime[indp] > n){
            if(n > 1)
                cout << n << '^' << 1;
            n = 1;
        }
    }
}

5. Verificarea Numerelor Prime

O funcție eficientă pentru verificarea primalității folosind vectorul de numere prime generat.

bool este_prim_ciur(int n){
    if(n == 0 || n == 1)
        return false;
    for(int i = 1; prime[i] * prime[i] <= n; ++i)
        if(n % prime[i] == 0)
            return false;
    return true;
}

6. Observații

  • Vectori Caracteristici: Ideali pentru verificări rapide.
  • Vectori de Frecvență: Eficienți pentru sortare și numărare.
  • Ciurul lui Eratostene: Complexitate O(n log n).
