Ștergeri și Inserări în Vectori

Operații pe Vectori

În această secțiune, vom explora funcții pentru ștergerea și inserarea elementelor într-un vector, incluzând operații pe elemente pare.

Codul complet:

#include <iostream>
using namespace std;

void stergere_element(int a[], int &n, int poz){
    for(int i = poz + 1; i <= n; ++i)
        a[i - 1] = a[i];
    n--;
}

void stergere_elemente_pare(int a[], int &n){
    for(int i = 1; i <= n; ++i)
        if(a[i] % 2 == 0){
            for(int j = i + 1; j <= n; ++j)
                a[j-1] = a[j];
            n--;
            i--;
        }
}

void inserare_element(int a[], int &n, int poz, int val){
    for(int i = n; i >= poz; --i)
        a[i+1] = a[i];
    a[poz] = val;
    n++;
}

void dublare_elemente_pare(int a[], int &n){
    for(int i = 1; i <= n; ++i)
        if(a[i] % 2 == 0){
            for(int j = n; j >= i + 1; --j)
                a[j+1] = a[j];
            a[i+1] = a[i];
            n++;
            i++;
        }
}

int main() {
    int a[1001], n;
    cin >> n;
    for(int i = 1; i <= n; ++i)
        cin >> a[i];

    dublare_elemente_pare(a, n);

    for(int i = 1; i <= n; ++i)
        cout << a[i] << ' ';

    return 0;
}

Explicații:

  • stergere_element: Elimină un element dintr-o poziție specificată a vectorului.
  • stergere_elemente_pare: Elimină toate elementele pare din vector.
  • inserare_element: Inserează un element la o poziție specificată.
  • dublare_elemente_pare: Dublează toate elementele pare prin inserarea unei copii adiționale imediat după ele.

Observații:

Operațiile implică modificări directe ale dimensiunii vectorului (n), iar acestea trebuie gestionate cu atenție pentru a evita accesarea elementelor invalide.
