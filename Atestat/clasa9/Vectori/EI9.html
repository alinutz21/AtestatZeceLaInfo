Vectori în C++

În această secțiune, vom explora utilizarea vectorilor în C++, incluzând operațiile de referențiere, dereferențiere și manipulare a elementelor.

Explicații teoretice

Operatori:
& - operator de referențiere: returnează adresa de memorie unde este stocat un element.
* - operator de dereferențiere: returnează valoarea stocată la o anumită adresă de memorie.

Relații importante:
a[i] <=> *(a + i) - valoarea din memorie a lui a[i].
&a[i] <=> a + i - adresa de memorie unde este stocat a[i].

Codul complet:

#include <iostream>
using namespace std;

int a[101]; // are 101 elemente declarate de la pozitia 0...100
int n; // dimensiunea sirului

int main(){

    a[0] = 7;
    a[1] = 5;
    a[2] = 12;

    cout << &a[0] << ' ' << &a[1] << ' ' << &a[2] << '\n';

    cout << *a << ' ' << *(a + 1) << ' ' << *(a + 2);

    cin >> n;
    for(int i = 0; i < n; ++i)
        cin >> a[i];

    for(int i = 0; i < n; ++i)
        cout << i[a] << ' '; 

    int *v = a;

    cout << v[0] << ' ' << v[1] << v[2];

    return 0;
}

Rezumat

  • a[0], a[1] și a[2] sunt primele 3 elemente ale vectorului.
  • Adresele de memorie ale elementelor vectorului pot fi obținute cu &a[i].
  • Valorile elementelor vectorului pot fi accesate cu *a sau *(a + i).
  • Perechea i[a] este echivalentă cu a[i], datorită simetriei în aritmetica pointerilor.

Notă:

Codul demonstrează cum pointerii pot fi utilizați pentru a lucra cu vectori, iar conceptul poate fi extins pentru alte structuri de date.
