Divizibilitate

Divizori ai unui număr

Un divizor al unui număr n este orice număr d care împarte n fără a lăsa rest. Pentru a determina toți divizorii unui număr, putem folosi mai multe metode:

Metoda 1: Iterație simplă

Verificăm pentru fiecare număr d de la 1 până la n dacă n % d == 0.


void afisare_divizori1(int n){
    for(int d = 1; d <= n; ++d)
        if(n % d == 0)
            cout << d << ' ';
}

Metoda 2: Până la rădăcina pătrată a lui n

Se iterează doar până la rădăcina pătrată a lui n, iar pentru fiecare divizor găsit, și n / d este un divizor.


void afisare_divizori2(int n){
    for(int d = 1; d * d <= n; ++d)
        if(n % d == 0 && d * d < n)
            cout << d << ' ' << n / d << ' ';
        else if(d * d == n)
            cout << d << ' ';
}

Număr Prim

Un număr n este prim dacă are exact doi divizori: 1 și el însuși.

Verificarea unui număr prim

Folosim o metodă de verificare a divizibilității pentru fiecare număr d până la rădăcina pătrată a lui n.


bool este_prim(int n){
    if(n <= 1)
        return false;
    for(int d = 2; d * d <= n; ++d)
        if(n % d == 0)
            return false;
    return true;
}

Cel Mai Mare Divizor Comuns (CMMDC)

Cel mai mare divizor comun al a două numere este cel mai mare număr care le divide pe ambele.

Algoritmul lui Euclid

Metoda clasică a lui Euclid pentru calcularea CMMDC-ului:


int cmmdc(int a, int b){
    while(a != b){
        if(a > b)
            a -= b;
        else b -= a;
    }
    return a;
}

Factorizare

Factorizarea unui număr în factori primi presupune descompunerea acestuia ca produs de factori primi.

Factorizare simplă


void factorizare(int n){
    int d = 2;
    while(n > 1){
        int p = 0;
        while(n % d == 0)
            n /= d, p++;
        if(p != 0)
            cout << d << '^' << p << '\n';
        d++;
        if(d * d > n)
            d = n;
    }
}

Teorema lui Legendre

Teorema lui Legendre se folosește pentru a calcula numărul de factori primi într-un număr factorial n! pentru un anumit factor prim.


int nr_zero_final_fact1(int n){
    int put2 = 0, put5 = 0;
    int d2 = 2, d5 = 5;
    while(d2 <= n){
        put2 += n / d2;
        d2 *= 2;
    }
    while(d5 <= n){
        put5 += n / d5;
        d5 *= 5;
    }
    return min(put2, put5);
}
