1. Permutări

Permutările reprezintă toate rearanjările posibile ale unei mulțimi cu n elemente.

Cod:

void back(int poz){
    for(int i = 1; i <= n; ++i){
        if(P[i] == 0){
            sol[poz] = i;
            P[i] = 1;
            if(poz == n)
                afis();
            else back(poz + 1);
            P[i] = 0;
        }
    }
}

Exemplu: Dacă n = 3, permutările sunt 1 2 3, 1 3 2, etc.
