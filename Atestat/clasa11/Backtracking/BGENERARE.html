Generarea Elementelor Combinatoriale

Explorăm tehnici pentru generarea permutărilor, aranjamentelor, combinărilor, submulțimilor și partițiilor unei mulțimi.

1. Permutări

Permutările reprezintă toate rearanjările posibile ale unei mulțimi cu n elemente.

Cod:

void back(int poz){
    for(int i = 1; i <= n; ++i){
        if(P[i] == 0){
            sol[poz] = i;
            P[i] = 1;
            if(poz == n)
                afis();
            else back(poz + 1);
            P[i] = 0;
        }
    }
}

Exemplu: Dacă n = 3, permutările sunt 1 2 3, 1 3 2, etc.

2. Aranjamente

Aranjamentele sunt submulțimi ordonate de k elemente dintr-o mulțime de n elemente.

Cod:

void back(int poz){
    for(int i = 1; i <= n; ++i){
        if(P[i] == 0){
            sol[poz] = i;
            P[i] = 1;
            if(poz == k)
                afis();
            else back(poz + 1);
            P[i] = 0;
        }
    }
}

Exemplu: Dacă n = 4, k = 2, aranjamentele sunt 1 2, 1 3, etc.

3. Combinări

Combinările reprezintă submulțimi neordonate de k elemente.

Cod:

void back(int poz){
    for(int i = sol[poz-1] + 1; i <= n - k + poz; ++i){
        sol[poz] = i;
        if(poz == k)
            afis();
        else back(poz + 1);
    }
}

Exemplu: Dacă n = 5, k = 3, combinările sunt 1 2 3, 1 2 4, etc.

4. Submulțimi

Generarea tuturor submulțimilor unei mulțimi cu n elemente.

Cod:

void back(int poz){
    for(int i = sol[poz-1] + 1; i <= n; ++i){
        sol[poz] = i;
        afis(poz);
        back(poz + 1);
    }
}

Exemplu: Dacă n = 3, submulțimile sunt {}, {1}, {1, 2}, etc.

5. Partitii

Partiționarea unei mulțimi în submulțimi disjuncte.

Cod:

void back(int poz, int nr_s){
    for(int i = 1; i <= nr_s + 1; ++i){
        sol[poz] = i;
        if(poz == n)
            afis(max(nr_s, i));
        else
            back(poz + 1, max(nr_s, i));
    }
}

Exemplu: Dacă n = 3, partițiile sunt {{1, 2, 3}}, {{1}, {2, 3}}, etc.

6. Observații

  • Toate problemele folosesc tehnica backtracking pentru explorarea soluțiilor posibile.
  • Generarea combinatorică este esențială în rezolvarea problemelor de optimizare și enumerare.
