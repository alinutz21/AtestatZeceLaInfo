Metoda Greedy

Selecția soluțiilor locale optime pentru rezolvarea problemelor complexe.

1. Introducere în metoda Greedy

Metoda Greedy (sau lacomă) presupune alegerea soluției optime la nivel local, în fiecare etapă a algoritmului, cu speranța că această strategie va conduce la o soluție globală optimă.

2. Exemplu: Problema Proiectelor

Avem o serie de proiecte reprezentate prin timpii de execuție: a, b, c, d, e. Metoda greedy sortează proiectele crescător și calculează timpul mediu alocat fiecărui proiect pentru a minimiza timpul total de așteptare:

Formula timpului mediu:

Tmediu = (5·a + 4·b + 3·c + 2·d + e) / 5

Codul corespunzător în C++ este:

#include 
using namespace std;

int main() {
    vector proiecte = {5, 3, 8, 6, 2};
    sort(proiecte.begin(), proiecte.end());
    
    int timp_total = 0, suma = 0;
    for (int timp : proiecte) {
        suma += timp;
        timp_total += suma;
    }

    cout << "Timpul mediu: " << timp_total / proiecte.size() << endl;
    return 0;
}

3. Problema Intervalelor

Se cere determinarea numărului maxim de intervale care nu se intersectează. Metoda greedy sortează intervalele după capătul din dreapta:

Codul pentru rezolvarea acestei probleme:

#include 
using namespace std;

struct Interval {
    int start, end;
};

bool cmp(Interval a, Interval b) {
    return a.end < b.end;
}

int main() {
    vector intervale = {{1, 2}, {2, 4}, {3, 5}, {6, 8}, {5, 9}};
    sort(intervale.begin(), intervale.end(), cmp);

    int count = 0, last_end = -1;
    for (auto &interval : intervale) {
        if (interval.start >= last_end) {
            count++;
            last_end = interval.end;
        }
    }

    cout << "Numărul maxim de intervale: " << count << endl;
    return 0;
}

4. Problema Sumei Minime

Avem un șir de numere și dorim să calculăm suma totală minimă combinând mereu cele mai mici două elemente. Costul fiecărei operații este suma elementelor combinate:

Codul pentru rezolvarea problemei:

#include 
using namespace std;

int main() {
    priority_queue, greater> pq;
    vector valori = {1, 2, 3, 4, 5};
    for (int val : valori) pq.push(val);

    int cost_total = 0;
    while (pq.size() > 1) {
        int a = pq.top(); pq.pop();
        int b = pq.top(); pq.pop();
        int cost = a + b;
        cost_total += cost;
        pq.push(cost);
    }

    cout << "Costul minim total: " << cost_total << endl;
    return 0;
}

5. Observații

  • Metoda Greedy funcționează optim atunci când problema satisface proprietatea de substructură optimă.
  • Sortarea este un pas crucial pentru multe probleme rezolvate cu această metodă.
  • Deși metoda poate da soluții suboptimale pentru unele probleme, este extrem de eficientă în practică.
